# 예술적 시드 그래프 중심 확장 계획

사용자가 직접 드로잉하기 전에, 푸리에 변환의 경이로움을 즉시 느낄 수 있는 **고품질 수학적 시드 그래프**를 우선적으로 제공합니다.

## 🎨 선정된 시드 그래프 프리셋

### 1. 하모닉 리사주 (Harmonic Lissajous)

- **수학적 정의**: $x = A \sin(at + \delta), y = B \sin(bt)$
- **비주얼**: 여러 주파수가 조화롭게 얽히며 만들어내는 기하학적 매듭 형태. '수학적인 우아함'을 가장 잘 보여줍니다.

### 2. 하이테크 스피로그래프 (Hypotrochoid)

- **수학적 정의**: 원 안의 원이 구르며 그리는 궤적.
- **비주얼**: 복잡한 꽃 문양이나 기하학적 문양(Flower of Life 느낌)을 생성하며, 푸리에 원들이 이 복잡한 문양을 완벽하게 재구성하는 과정이 매우 화려합니다.

### 3. 푸리에 피닉스/심볼 (Fourier Phoenix)

- **정의**: 미리 정의된 복잡한 벡터 데이터 (예: 무한대 기호 또는 특정 문양).
- **비주얼**: 단순한 기하학을 넘어, 수백 개의 원이 모여 하나의 정교한 '그림'을 완성하는 퍼포먼스를 보여줍니다.

## ✅ 검증 계획

### 시각적 및 성능 테스트

- 10개 이상의 멀티 푸리에 객체가 동시에 구동될 때 60FPS 유지 여부.
- 시드 그래프의 수학적 형태가 정확하게 복원되는지 확인.

1. 각 시드 그래프가 선택되었을 때, 지연 없이 애니메이션이 시작되는지 확인.
2. 수학 공식에 기반한 좌표 생성 시 앨리어싱(Aliasing) 현상이 없는지 샘플링 밀도 점검.
